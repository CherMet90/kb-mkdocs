# Харденинг SSH на Cisco ASR 1000 (IOS-XE)

## Терминология

### VTY (Virtual Teletype)

**VTY** — это виртуальные терминальные линии, предназначенные для удалённого управления устройством по протоколам SSH, Telnet или Rlogin. Каждая активная сессия администратора занимает одну VTY-линию. Количество линий (`line vty 0 4`, `0 30` и т.д.) определяет максимальное количество одновременно подключённых пользователей.

Доступ к VTY-линиям может быть ограничен через:

- **access-class** — ACL, фильтрующий входящие соединения по IP-адресу источника.
- **transport input** — протокол, разрешённый для подключения (рекомендуется только `ssh`).

### CoPP (Control Plane Policing) — общий принцип

CoPP (Control Plane Policing) — это механизм QoS-фильтрации, применяемый к трафику, предназначенному **control plane** (CPU) устройства. Он позволяет классифицировать такой трафик (основные протоколы — SSH, SNMP, BGP, OSPF, ICMP, ARP) и применить к нему полисер с действием:

- **transmit** — пропускать в пределах лимита;
- **drop** — отбрасывать излишек.

**Задачи, которые решает CoPP:**

- Защита CPU от DoS-атак (флуд на control plane может вызвать 100% загрузку CPU).
- Ограничение частоты (rate-limit) протоколов управления для предотвращения resource exhaustion.
- Освобождение ресурсов CPU для критического контроля плоскости (маршрутизация, сигнализация).
- Дополнительный рубеж защиты после ACL (defence-in-depth).

## Базовые параметры SSH и отключение веб-интерфейса

```cisco
no ip http server
no ip http secure-server
!
ip ssh time-out 15          ! Таймаут на установление сессии
ip ssh version 2
```

**Важно:** Отключение HTTP/HTTPS убирает дополнительные векторы атак на management plane и исключает ложные срабатывания `login block-for` от веб-сканеров, генерирующих мусор в логах авторизации.

## Настройка VTY ACL

```cisco
ip access-list standard ACL-VTY
remark === hosts ===
permit 10.20.1.10
remark === OOB net ===
permit 10.50.1.0 0.0.0.255      ! подсеть mgmt-интерфейса
permit 10.50.2.0 0.0.0.255      ! другие подсети, имеющие доступ к mgmt-интерфейсу
remark === admin nets ===
permit 10.100.10.0 0.0.0.255
permit 10.100.20.0 0.0.0.255
permit 10.100.30.164 0.0.0.1     ! пример маски /31 в wildcard
permit 10.100.30.248 0.0.0.3     ! пример маски /30 в wildcard
permit 172.16.10.0 0.0.0.255
permit 198.51.100.0 0.0.0.255
deny   any log                   ! Логируем отклоненные попытки
!
line vty 0 30
access-class ACL-VTY in vrf-also
transport input ssh
```

**Ограничение:** Стандартный ACL не фильтрует по L4-порту. Это допустимо для VTY, так как `line vty` сама принимает только терминальные протоколы, но при переиспользовании `ACL-VTY` в других местах учитывайте отсутствие фильтрации по порту.

**Важно:** Параметр `vrf-also` обязателен при использовании Management VRF для OOB-доступа. Без него подключения через интерфейсы `Mgmt-intf` будут блокироваться на уровне IOS, независимо от правил внутри ACL.

## Настройка CoPP для SSH

```cisco
! ACL матчит SSH в обе стороны (защита + ответы)
ip access-list extended COPP-SSH
remark === SSH to/from router ===
permit tcp any any eq 22
permit tcp any eq 22 any
!
class-map match-any COPP-SSH
match access-group name COPP-SSH
!
policy-map COPP-POLICY
class COPP-SSH
police rate 200 pps burst 100 packets conform-action transmit exceed-action drop
```

**Важно:** Класс `COPP-SSH` автоматически встраивается в `policy-map` **перед** `class-default`. Специфичные классы должны идти выше общего catch-all, иначе трафик будет поглощён `class-default` без ограничения.

**Ограничение:** При вставке больших конфигурационных портянок (>50 строк) возможен временный рост счётчика `exceeded` из-за превышения burst. Это приводит к замедлению отрисовки ввода, но не к потере команд: TCP гарантированно выполнит retransmit

## Верификация

```bash
show policy-map control-plane | section COPP
```

**Примечание:** Обращайте внимание на поле `exceeded`. Нулевое или незначительно растущее значение в штатном режиме подтверждает корректность лимита. Одновременный резкий рост `conformed` и `exceeded` указывает на сканирование или флуд, который успешно отсекается политикой.
